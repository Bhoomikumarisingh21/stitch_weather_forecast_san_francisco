# 🌤️ Atmospheric Precision - Weather Forecast App

A premium, clean, and highly responsive Weather Forecasting application built using **Flutter** and **Dart**. This app offers real-time meteorological insights packed into a stunning user interface that dynamically adapts to environmental conditions.

---

## 🔥 Key Features

* **🌍 Global City Search:** Search and jump to any city worldwide (e.g., London, San Francisco) to instantly view its current weather.
* **⏳ 24-Hour Hourly Forecast:** Track a detailed hour-by-hour breakdown of temperature and weather conditions to plan your day seamlessly.
* **🔄 Smart Unit Converter:** Flexible toggles to switch between Metric and Imperial systems instantly (Celsius $\leftrightarrow$ Fahrenheit, km/h $\leftrightarrow$ mph).
* **🎨 Immersive Premium Themes:** Beautiful custom themes curated for visual comfort—switch smoothly between *Rainy Gray*, *Sunset Gold*, and *Deep Dark* modes.
* **🖼️ Dynamic Visuals:** The background and layout elements react responsively based on the selected location and weather state.

---

## 🛠️ Tech Stack & Tools Used

* **Frontend Framework:** Flutter 
* **Programming Language:** Dart
* **Data Layer:** Real-time Weather API integration for accurate, live data syncing.
* **IDE:** Antigravity / VS Code

---

## 💡 Why is this App Useful?

1. **Daily Planning:** Users can check precise hourly conditions before heading out, ensuring they never get caught unprepared in bad weather.
2. **Travel Companion:** Perfect for tracking destination climates in advance to streamline travel and packing plans.
3. **Production-Ready Showcase:** Demonstrates real-world implementation of asynchronous data fetching (APIs), dynamic state management, responsive multi-platform UI design, and native compilation for both Android (APK) and Web.
